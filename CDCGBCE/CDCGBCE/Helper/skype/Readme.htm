條碼列印異常說明

 

因使用者電腦安裝Skype軟體的關係，Skype軟體會將09xxxxx-xx的字樣自動轉
成電話號碼，導致條碼列印出現異常，如下圖：

 

解決方式：

一、於桌面上左下角點選『開始』→『設定』→『控制台』

 

二、進入控制台後，點選「網際網路選項」

 

三、選擇上方「程式集」選項，點選下方「管理附加元件」

 

四、進入管理附加元件後，找到名稱為「Skype」，點選下方設定中之「停用」

 

五、找到名稱為「Skype add-on」，點選下方設定中之「停用」後點選右下方之
「確定」。

 

六、關閉所有的IE瀏覽器或Skype軟體再開啟，即可印出正確報表條碼格式。
